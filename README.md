# FocusTab

FocusTab は、新規タブを集中ポータルに置き換え、仕事・学習・趣味などのモードに合わせてタブ・タスク・ブックマーク・閲覧制限を一括で切り替える Chrome 拡張機能です。データはすべてブラウザ内に保存され、アカウント登録やクラウド同期は不要です。

## 主な機能

| 機能                       | 説明                                                                                       |
| -------------------------- | ------------------------------------------------------------------------------------------ |
| **モード切替**             | 現在のタブを退避し、モードに保存されたタブを復元。テーマカラーも切り替わります             |
| **閲覧制限**               | 制限モードではブラックリストのドメインへのアクセスをブロック（DNR + SPA 補完）             |
| **タスク管理**             | モードごとのタスクを追加・編集・完了・削除。ドラッグで並び替え。完了タスクはアーカイブ可能 |
| **ブックマーク**           | モードごとのクイックリンクを管理（編集・ドラッグで並び替え可能）                           |
| **モードロック**           | 15 / 30 / 60 / 120 分のプリセットでモード切替をロック                                      |
| **タスクアーカイブ**       | 完了タスクをアーカイブして一覧から非表示に（復元・一括削除可能）                           |
| **ツールバーポップアップ** | 現在モードの確認、モード切替、ポータル・設定へのクイックアクセス                           |
| **緊急解除**               | ブロック画面から固定文の入力でモードロックを解除（サイトの閲覧制限は続く）                 |
| **モード管理**             | カスタムモードの作成・編集・削除（最大 10 モード）                                         |
| **データのバックアップ**   | JSON 形式でエクスポート / インポート                                                       |

## インストール

### 開発ビルドから読み込む

```bash
git clone <このリポジトリの URL>
cd FocusTab
npm install
npm run build
```

1. Chrome で `chrome://extensions` を開く
2. 右上の **デベロッパーモード** をオンにする
3. **パッケージ化されていない拡張機能を読み込む** をクリック
4. ビルド出力フォルダ `.output/chrome-mv3` を選択

### 開発中（HMR 付き）

```bash
npm install   # 初回または node_modules がない場合
npm run dev   # 開発サーバー起動（localhost:3000）が必須
```

1. 上記コマンドで開発サーバーを起動したままにする
2. Chrome で `.output/chrome-mv3-dev` を読み込む（本番ビルドの `chrome-mv3` ではない）
3. コード変更は自動で反映されます

### 配布用 ZIP

```bash
npm run zip
```

`.output/focustab-<version>-chrome.zip` が生成されます。

## 使い方

### 初回起動

新しいタブを開くと FocusTab ポータルが表示されます。初回は 3 ステップのオンボーディングが表示され、機能概要とプライバシー方針を説明します。

### モード切替

ヘッダー下のモードボタンで切り替えます。

- 切替時、**現在開いているタブは退避**され、対象モードに **保存済みのタブが復元** されます
- 設定で「モード切替の確認」をオンにしている場合、確認ダイアログが表示されます
- **モードロック中** は切替できません

### タスク・ブックマーク

| パネル                       | 操作                                                                                                                                                               |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **タスク**（左カラム）       | テキスト入力で追加、「編集」で文言変更、チェックで完了、× で削除。未完了タスクはドラッグで並び替え。完了タスクはアーカイブ可能（アーカイブ一覧から復元・一括削除） |
| **ブックマーク**（右カラム） | URL とタイトルを登録・編集。ドラッグで並び替え。リンクをクリックで開く                                                                                             |

タスクとブックマークは **アクティブなモードごと** に独立して管理されます。

### モードロック

右カラムの「モードロック」から 15 / 30 / 60 / 120 分を選択します。ロック中はモード切替ができず、残り時間がカウントダウン表示されます。

### 閲覧制限とブロック画面

制限付きモード（`isRestrictive: true`）では、ブラックリストに登録したドメインへのアクセスがブロックされます。ブロック時は専用ページが表示され、「タスク一覧へ」でポータルに戻れます。

SPA（シングルページアプリ）のクライアントサイド遷移も `tabs.onUpdated` で補完しています。

### 緊急解除（モードロック解除）

モードロック中に制限サイトへアクセスするとブロック画面が表示されます。右下の **モードロック解除** から、以下の固定文を **正確に手入力** してください（コピー＆ペーストは無効）。

> 集中を中断すると今日の目標が遠のきます。本当に解除しますか？

入力が正しければ **モードロックのみ** 解除されます。**このサイトへの閲覧制限は続きます。** 別のモードに切り替えるには「タスク一覧へ」からポータルに戻ってください。

ロックしていない場合は緊急解除ボタンは表示されません。

### モード管理

ヘッダーの **モード管理** からカスタムモードを作成・編集・削除できます。

- モード名、アクセントカラー、制限の有無を設定
- 制限モードではブラックリスト（ドメインのみ、1 行 1 ドメイン）を登録
- デフォルトの 3 モード（仕事・学習・趣味）も編集可能
- **初期モードを復元** で仕事・学習・趣味を初期設定に戻せる（カスタムモードは保持）
- アクティブなモードは削除できません

### ツールバー・オプション

- 拡張機能アイコンをクリック → **ポップアップ**（現在モード・ロック状態、モード切替、ポータル / 設定へのボタン）
- `chrome://extensions` の **拡張機能のオプション** → 設定付き新規タブを開く

### 設定・バックアップ

ヘッダーの **設定** から以下を変更できます。

| 設定項目             | 説明                                   |
| -------------------- | -------------------------------------- |
| モード切替の確認     | 切替前に確認ダイアログを表示           |
| タブ復元の進捗表示   | 復元中のプログレスバナーを表示         |
| 復元バッチサイズ     | 一度に開くタブ数（1〜5、既定 3）       |
| データのエクスポート | 全データを JSON ファイルでダウンロード |
| データのインポート   | JSON ファイルで全データを上書き復元    |

## プライバシーと権限

FocusTab は **ローカルファースト** です。タスク・ブックマーク・タブスナップショット・設定はすべて `chrome.storage` に保存され、外部サーバーへ送信されません。

| 権限                                  | 用途                                 |
| ------------------------------------- | ------------------------------------ |
| `tabs`                                | モード切替時のタブ退避・復元         |
| `storage`                             | 設定・タスク・ブックマーク等の永続化 |
| `alarms`                              | モードロックの期限管理               |
| `declarativeNetRequest`               | ブラックリストドメインのブロック     |
| `declarativeNetRequestWithHostAccess` | 全 URL に対する DNR ルール適用       |
| `webNavigation`                       | ナビゲーション時のブロック判定       |
| `<all_urls>`                          | 上記 DNR / ナビゲーション監視        |

ページの内容を読み取ったり、閲覧履歴を外部に送信したりすることはありません。

## 制限事項

- **対応ブラウザ**: Chromium 系（Chrome / Edge 等）Manifest V3
- モードあたり最大 **100 タスク**、**50 ブックマーク**、**200 ブラックリストドメイン**
- タブスナップショットはモードあたり最大 **30 タブ**
- カスタムモードは最大 **10 個**
- 閲覧制限はドメインベースです。VPN や別ブラウザでは回避可能です（自己規律ツールとしての位置づけ）
